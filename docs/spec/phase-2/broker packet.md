## C. The formal WCIRB classification question (written determination requested)

> **"For a commercial employer (not a direct householder) dispatching W-2 employees into clients' homes for a hybrid mix of in-home childcare, cleaning, and meal preparation for typically developing children, what is the governing WCIRB classification? If no single code cleanly applies, may payroll be split between 9096 (cleaning) and another code for the childcare/meal-prep portion, and if so, which code and under what recordkeeping standard?"**

Groundwork already verified against WCIRB USRP text, so the submission can be precise:
- **8827(1) Home Care Services** is expressly scoped to customers needing services **"due to age, illness or disability"** (incl. SLS/IHSS); its own phraseology routes stand-alone residential cleaning to **9096**. Routine care of typically developing children appears **outside** 8827 — please do not force-fit it there without WCIRB's written concurrence.
- **9059 Day Care Centers – Child** is facility/program-based on its face; it was not written for dispatching individual caregivers into private homes.
- **No dedicated in-home childcare-dispatch code** was found in the USRP or DIR class listings — hence the formal inquiry.
- **Payroll splitting:** USRP Part 3, §V, Rule 3 — contemporaneous records, "summarized... by operation"; **no percentages, averages, or estimates**; absent a valid split, the entire payroll defaults to the highest-rated applicable classification. We understand task-coded daily summaries suffice for non-construction classes (minute-level punches being a dual-wage construction requirement) — please confirm the carrier/auditor will honor that format.
- **Lab. Code §3351(d)** private-residence carve-out does not apply (the cooperative, not the household, is the employer).
